# WA AI Helper

Telegram-бот для менеджеров ВЭД-компании West Asia. Расчёт таможенных платежей, проверка ТН ВЭД, конвертация валют (CNY, USD, EUR), курсы ЦБ РФ, база знаний.

---

## 1. Назначение

**West Asia AI Helper** — Telegram-бот (aiogram 3.x + DeepSeek API + SQLite) для менеджеров ВЭД-компании West Asia. Помогает быстро рассчитывать таможенные платежи, определять ставки по ТН ВЭД, конвертировать валюты (CNY, USD, EUR) и структурировать заявки на перевозку — всё в одном окне чата.

---

## 2. Кому помогает

Менеджерам по ВЭД и логистике, которые:

* Ввозят грузы в Россию из любой точки мира (основное направление — Китай)
* Работают по схеме А (логистика + декларирование под контракт клиента) или схеме Б (договор поставки под ключ с выкупом груза)
* Нуждаются в мгновенной оценке стоимости груза с учётом пошлин, НДС и сборов

---

## 3. Что умеет (функционал)

### 3.1 Расчёт таможенных платежей (основной сценарий)

Пользователь пишет текст с данными груза → бот возвращает структурированный расчёт из трёх блоков: шапка, конвертация, итог.

**Исходные данные (шапка):**

* Код ТН ВЭД с полным наименованием (из загруженного Excel tws.by)
* Ставка пошлины (адвалорная / комбинированная / специфическая) с формулой
* НДС: 22% базовая или 10% льготная (продовольствие, детские, медицина, печать)
* Таможенный сбор по шкале ПП РФ №1637 или фиксированный сбор за радиоэлектронику (73 860 ₽ для 106 кодов)

**Конвертация в валюту инвойса (CNY / USD / EUR):**

* Все компоненты (фрахт, страховка, упаковка) конвертируются в валюту инвойса через курс ЦБ РФ
* Рубль — промежуточная валюта: CNY→USD = CNY/₽ ÷ USD/₽, CNY→EUR = CNY/₽ ÷ EUR/₽
* Сбор показывается в двух валютах: `1 231 ₽ → 132.48 CNY` или `1 231 ₽ → 18.50 EUR`
* Поддерживаются инвойсы в CNY, USD и EUR

**Пример вывода:**

```
📋 Код: 8471300000
🔧 Портативные автоматические вычислительные машины весом не более 10 кг
💰 Пошлина: 0% — адвалорная
🧾 НДС: 22% (базовая)
⚡ Радиоэлектроника: сбор 73 860 ₽ (код 8471 в Приложении №1)

🔄 Конвертация в валюту инвойса (CNY):
• Инвойс: 100 000 CNY — уже в валюте инвойса
• Фрахт: 900 USD → 65 814.75 ₽ → 6 132.48 CNY
• Страховка: 2 500 ₽ → 232.95 CNY

📊 Итоговый расчёт
💰 Таможенная стоимость: 106 365.43 CNY
📋 Пошлина 0%:                0.00 CNY
🧾 НДС 22%:              23 400.39 CNY
⚡ Сбор:                  6 896.52 CNY  (73 860 ₽ → 6 896.52 CNY)
─────────────────────────────────────────
💵 ИТОГО:                30 296.91 CNY  (~ 324 500.00 ₽)

📌 Точную информацию уточняйте у декларанта.
💱 Курс ЦБ РФ на 18.05.2026: 1 USD = 73.1264 ₽, 1 CNY = 10.7345 ₽, 1 EUR = 92.5687 ₽
```

**Правила конвертации:**

* Инвойс — не конвертируется (уже в валюте инвойса)
* Компонент в другой валюте → ₽ по курсу ЦБ РФ → валюта инвойса
* Компонент уже в ₽ → напрямую в валюту инвойса
* Компонент уже в валюте инвойса — оставляем как есть

**Примеры кросс-конвертации:**

| Из → В | Формула |
|--------|---------|
| USD → CNY | USD × USD/₽ ÷ CNY/₽ |
| USD → EUR | USD × USD/₽ ÷ EUR/₽ |
| EUR → CNY | EUR × EUR/₽ ÷ CNY/₽ |
| ₽ → CNY | ₽ ÷ CNY/₽ |
| ₽ → EUR | ₽ ÷ EUR/₽ |

**Ключевые правила итогового расчёта:**

* Все суммы в блоке `📊 Итоговый расчёт` — **только в валюте инвойса**
* Сбор (таможенный или за радиоэлектронику) — конвертируется из ₽ в валюту инвойса
* Рубли `~ XXXXXX ₽` после `ИТОГО` — **справочно**, для ориентира
* Курс ЦБ РФ — всегда в конце: CNY, USD, EUR

---

### 3.2 Быстрый ответ (только код ТН ВЭД)

Если в запросе нет сумм — бот выдаёт шапку с кодом, наименованием, пошлиной, НДС и пометкой о радиоэлектронике без расчёта.

---

### 3.3 Расчёт из Excel (tws.by)

* Загрузка `.xlsx` со справочником ТН ВЭД через `/updatecodes`
* Парсинг ставок: адвалорные (X%), комбинированные (X% + Y EUR/кг), специфические (X USD/кг или X EUR/кг)
* Fallback на внешние сайты отключён — только данные из Excel
* Полные наименования ТН ВЭД — автозаполнение `TNVED_FULL_NAMES` из загруженного Excel

---

### 3.4 Курс ЦБ РФ

* Автоматическое получение курсов CNY, USD, EUR с cbr.ru
* Курс добавляется в каждый ответ: `💱 Курс ЦБ РФ на 18.05.2026: 1 USD = 73.1264 ₽, 1 CNY = 10.7345 ₽, 1 EUR = 92.5687 ₽`

---

### 3.5 Страхование груза (п. 1 ст. 40 ТК ЕАЭС)

* Страховая премия включается в таможенную стоимость
* Страхование НЕ выводится отдельной строкой в итоге — оно внутри ТС
* В конвертации показывается: `Страховка: 2 500 ₽ → 232.95 CNY`

---

### 3.6 Бриф на проект / груз

Команда `/brief` выдаёт структурированную форму (6 секций):

1. Контактные данные
2. Товар (наименование, ТН ВЭД, количество, страна происхождения)
3. Стоимость (инвойс, Incoterms, фрахт, страхование)
4. Схема работы (А или Б)
5. Маршрут (откуда, куда, вид транспорта, сроки)
6. Дополнительно (температурный режим, опасный груз, негабарит)

---

### 3.7 База знаний (/learn)

* Администратор может обучать бота новым темам через текст или файлы (.txt, .docx, .xlsx)
* Вопросы для проверки понимания генерируются автоматически

---

### 3.8 Экспорт логов

* `/логи` — выгрузка диалогов в Excel (.xlsx) с фильтрацией по дате

---

### 3.9 Обратная связь

* Ответ «несогласен» на сообщение бота записывает замечание и уведомляет администратора

## 4. Ключевые бизнес-правила

| Правило | Значение |
|---------|----------|
| НДС базовая | 22% (с 01.01.2026) |
| НДС льготная | 10% (продовольствие, детские, медицина, печать) |
| Таможенный сбор | Шкала ПП РФ №1637: до 200K→1 231 ₽, 450K→2 462 ₽, 1.2M→4 924 ₽, 2.7M→13 541 ₽, 4.2M→18 465 ₽, 5.5M→21 344 ₽, 10M→49 240 ₽, свыше→73 860 ₽ |
| Радиоэлектроника | Фиксированный сбор 73 860 ₽ (106 кодов: 84xx, 85xx, 90xx, 91xx, 95xx) |
| Валюта расчёта | Валюта инвойса (CNY / USD / EUR) — все суммы ТОЛЬКО в ней |
| Конвертация | Через рубль ЦБ РФ (CNY, USD, EUR), НИКОГДА не придумывать курсы |
| Страхование | Входит в ТС (не выводится отдельно) |
| Комбинированная ставка | Считать оба варианта (процент + EUR/кг), брать МАКСИМУМ |
| Вес неизвестен | Считать только процент, предупредить |

## 6. Архитектура (8 модулей)

| Файл | Назначение |
|------|------------|
| `config.py` | Константы, ENV, SYSTEM_PROMPT, CURRENCY_SYNONYMS (CNY/USD/EUR/RUB) |
| `bot_instance.py` | Bot + Dispatcher (aiogram) |
| `database.py` | SQLite: инициализация, 6 таблиц, экспорт Excel |
| `handlers.py` | Команды (/start, /help, /clear, /brief, /learn, /updatecodes) + текст |
| `utils.py` | Курсы ЦБ РФ (CNY/USD/EUR), DeepSeek API, rate limit, конвертация сборов |
| `parsers.py` | XLSX/txt/docx парсеры + парсинг тарифов ТН ВЭД |
| `tnved_engine.py` | Кэш ТН ВЭД в памяти, поиск, радиоэлектроника, полные наименования |
| `calc_engine.py` | Расчёт ТП, форматирование итоговой таблицы, strip-дубли DeepSeek |

```

├── main.py              # Точка входа
```

## 7. Критерии качества ответа

1. Нет дублей — шапка (код, пошлина, НДС) выводится один раз, DeepSeek не дублирует
2. Нет markdown-таблиц — только форматирование пробелами и эмодзи
3. Нет слов «сводка», «копируемая» — чистый вывод
4. Все суммы в валюте инвойса (CNY/USD/EUR) — включая сбор
5. Курс ЦБ РФ (CNY/USD/EUR) в каждом ответе — справочно в конце
6. Дисклеймер декларанта — `📌 Точную информацию уточняйте у декларанта`
7. Страхование в ТС — не выводится отдельной строкой
8. Не придумывать данные — курсы и ставки только из контекста/Excel
