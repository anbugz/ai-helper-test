# AI Helper West Asia

Telegram-бот для менеджеров по ВЭД и логистике. Расчёт таможенных платежей, проверка ТН ВЭД, курсы ЦБ РФ, база знаний.

## Стек

- **aiogram 3.7+** — Telegram Bot API
- **DeepSeek API** — LLM для расчётов
- **SQLite** — хранение диалогов, кэш ТН ВЭД, база знаний
- **httpx** — запросы к API (нет aiohttp на shared-хостинге)

## Структура

```
ai-helper-test/
├── main.py              # Точка входа
├── config.py            # ENV, константы, SYSTEM_PROMPT
├── bot_instance.py      # aiogram Bot + Dispatcher
├── database.py          # SQLite: 6 таблиц + TNVED кэш
├── parsers.py           # Парсинг Excel/DOCX/TXT + тарифы ТН ВЭД
├── tnved_engine.py      # Кэш ТН ВЭД в памяти, проверка радиоэлектроники
├── calc_engine.py       # Расчёт ТП, форматирование итоговой таблицы ← добавлен
├── utils.py             # Курсы ЦБ, safe_send, DeepSeek, rate_limit
├── handlers.py          # Все @dp.message хэндлеры
├── requirements.txt     # Зависимости (aiogram, openai, httpx) ← добавлен
├── .env.example         # Шаблон переменных окружения
└── .gitignore           # bot.db, __pycache__, .env
```

## Таблицы SQLite

| Таблица | Данные |
|---------|--------|
| `dialogs` | Все вопросы/ответы |
| `corrections` | Замечания («несогласен», «неверно») |
| `custom_radio_codes` | Загружаемые коды радиоэлектроники |
| `settings` | Настройки |
| `knowledge_base` | Темы обучения |
| `tnved_cache` | Кэш справочника ТН ВЭД (переживает редеплой) |

## Триггеры (без команд)

- **Текст** — AI-расчёт с проверкой ТН ВЭД в кэше
- **.xlsx файл** — загрузка справочника ТН ВЭД в кэш
- **«несогласен / неверно / неправильно»** — запись замечания, уведомление АБ

## Настройка

```bash
# 1. Клонировать
git clone https://github.com/USER/ai-helper-test.git
cd ai-helper-test

# 2. Создать .env
cp .env.example .env
# Отредактировать .env — вставить BOT_TOKEN, DEEPSEEK_API_KEY, ADMIN_ID

# 3. Запустить
python main.py
```

## Обновление (база не теряется)

```bash
cd ai-helper-test
git pull origin main
python main.py
```

База `bot.db` хранится вне папки проекта (`../bot-data/`) и не затрагивается при `git pull`.

## Критичные правила

- НДС: **22%** с 01.01.2026, льготная 10%
- Таможенный сбор: шкала ПП РФ №1637 (в ред. №1638)
- Радиоэлектроника: **73 860 ₽** фикс (106 кодов) → конвертируется в валюту инвойса
- Валюта: валюта инвойса (все платежи, включая сбор)
- Кросс-конвертация: через рубль ЦБ РФ
- TNVED_FULL_NAMES: автозаполнение из загруженного Excel
